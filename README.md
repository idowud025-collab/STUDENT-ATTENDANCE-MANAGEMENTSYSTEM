# Student Attendance Management System

## Introduction
The Student Attendance Management System is a Python-based application developed
for the SEN 201 course. It helps lecturers record and manage student attendance.

## Objectives
- To record student attendance
- To display attendance records
- To demonstrate SDLC concepts

## System Features
- Add student attendance
- View attendance records
- Menu-driven interface

## System Requirements
- Python 3.x
- Any operating system

## How to Run the Program
1. Clone or download the repository
2. Navigate to the project directory
3. Run the command:
   python student_attendance_management_system.py

## SDLC Applied
- Requirement Analysis
- System Design
- Implementation
- Testing
- Deployment
- Maintenance

## Limitations
- Data is not stored permanently
- Single-user system

## Future Enhancements
- File/database storage
- Graphical user interface
